### Dependencies

- [Glad 2](https://github.com/Dav1dde/glad)
- [OpenGL Mathematics (GLM)](https://github.com/g-truc/glm)
- [Simply DirectMedia Layer (SDL2)](https://wiki.libsdl.org/SDL2/Installation)
- [Miniply](https://github.com/vilya/miniply)  

SDL2 is not included in this repo. Find instructions for installing it on your OS [here](https://wiki.libsdl.org/SDL2/Installation).  
All other dependencies are located in `point-cloud-renderer/lib`.
